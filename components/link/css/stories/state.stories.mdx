## Focus-visible state

Styling met de `.utrecht-link--focus-visible` class naam of als status op `.utrecht-link:focus-visible`:

Het is belangrijk dat de `focus-visible` staat duidelijk herkenbaar is (_distinguishable_). Een link met `hover` en een link met `focus-visible` moeten een ander design hebben.

Een link met `focus-visible` moet niet herkenbaar alleen zijn omdat die er anders uit ziet in een groep van links. Ook als een link met `focus-visible` de enige zichtbare link is, moet het duidelijk zijn dat de link focus heeft. De link alleen groter maken is bijvoorbeeld onvoldoende, omdat dit alleen herkenbaar is in een groep van links.

    name="focus-visible"

</Canvas>
