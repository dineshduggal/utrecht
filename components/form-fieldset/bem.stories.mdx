# Fieldset

Styling via the `.utrecht-form-fieldset` and `.utrecht-form-fieldset__legend` class names.

The `--html-fieldset` and `--html-legend` modifiers help to reset browser default styling, and will include CSS properties that would not be necessary if a `div` element was used.
